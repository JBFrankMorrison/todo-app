# 📝 Todo App

Aplicación web para gestionar tareas, desarrollada con JavaScript puro.

## 🚀 Funcionalidades
- Agregar tareas
- Eliminar tareas
- Marcar como completadas
- Persistencia con localStorage

## 🛠 Tecnologías
- HTML
- CSS
- JavaScript
